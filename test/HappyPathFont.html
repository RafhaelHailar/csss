Notes:
  1. $font() - family, size?, weight , text align because inherited
  2. $text() - text-transform, text-indent, stretch
  3. $bold, $italic, $oblique(number)
  4. $spacing() - letter spacing, word spacing, line-height
  5. $decoration() - text-decoration-style, text-decoration-line, text-decoration-thickness
  6. $fontvar() - font-variant-caps, font-variant-east-asian, font-variant-ligatures, font-variant-numeric.
    • East Asian could be used in language
  7. $language() - line break, direction, writing-mode, text-combine-upright, text-orientation, text-transform
    • word-break is also used in language but can also be used in english so this should be a standalone dollar short?
    • direction is also used in language but can also be used in flex rows and table columns so I feel this should be a standalone dollar short? plus for japanese, writing mode is rather used.
    • vertical-align is not related to language but it gets affected when you set a language in different direction.
    • text-transform is generally used in other stuff too but it has properties that can be added in language. so maybe this can a property that is grouped in two things text() and language()?
  8. For future maybe because they are not widely supported: text-wrap&hypens
